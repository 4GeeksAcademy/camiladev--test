🌿Landing Page – Restaurante Vegano

Este proyecto consiste en el desarrollo de una landing page para un restaurante vegetariano/vegano, con enfoque en alimentación saludable, ética animal y productos orgánicos.

Actualmente se encuentra en etapa de desarrollo.

🚧 Estado del proyecto

En progreso.

Hasta el momento:

✔️ Estructura HTML semántica inicial
✔️ Organización de secciones principales (hero, menú, contacto)
⏳ Estilos CSS en desarrollo

Practicar y consolidar conocimientos de:

HTML5 semántico
CSS (Grid y Flexbox)
Diseño responsive
Buenas prácticas de estructura y organización
